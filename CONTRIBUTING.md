### 1. Fork & Clone

```bash
git clone https://github.com/YOUR_USERNAME/fathuptime.git
cd fathuptime
npm install
```

### 2. Buat Branch Baru

```bash
git checkout -b feature/nama-fitur-anda
```

### 3. Lakukan Perubahan

- Tulis kode yang bersih dan mudah dibaca
- Gunakan TypeScript dengan strict mode
- Follow existing code style
- Tambahkan komentar dalam Bahasa Indonesia

### 4. Test Perubahan

```bash
npm run dev
# Test manual di browser
```

### 5. Commit & Push

```bash
git add .
git commit -m "feat: menambahkan fitur X"
git push origin feature/nama-fitur-anda
```

### 6. Buat Pull Request

Buat PR dengan deskripsi yang jelas tentang perubahan yang Anda buat.
